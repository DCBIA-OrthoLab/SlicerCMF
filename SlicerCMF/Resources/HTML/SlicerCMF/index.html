SlicerCMF modules:

ShapePopulationViewer

This module allows to interact with multiple 3D surfaces at the same time. It supports visualization and comparison of 3D surfaces by displaying the associated pointwise data (scalar or vector maps) via customizable colormaps.

ShapeVariationAnalyzer

This module allows the classification of 3D models, according to their morphological variation. This tool is based on a deep learning neural network. The module is composed of multiple panels to perform the different steps of the process: create the classification groups, compute their average shapes, train the classifier and classify shapes.

Angle Planes

This module is used to calculate the angle between two planes. The user selects already loaded planes or they can define a plane by using three landmarks.

Bone Texture

This module allows to interact with multiple 3D surfaces at the same time. It supports visualization and comparison of 3D surfaces by displaying the associated pointwise data (scalar or vector maps) via customizable colormaps.

Surface Registration

This module performs region based registration.

EasyClip

This module is used to clip and close one or several models according to a predetermined plane. Planes can be saved and reused.

MeshStatistics

This module computes descriptive statistics (min, max, avg, std, 5th per, 15th per, 15th per, 75th per, 85th per and 99th per) on data fields of a model or models. The statistics can be computed over predefined regions (selected with PickAndPaint) or the entire model.

Mesh to Label Map

This module can converts a model into a binary segmentation image volume.

Model to Model Distance

This module computes a point by point distance between two models.

Pick and Paint

This module selects a region of interest (ROI) in a model or models. The user selects a landmark and a number of vertices to define the size of the ROI, and this information gets propagated to the rest of the models in case of having multiple ones.

Q3DC

This module allows to perform head measurements used in craniofacial surgery (called Quantitative 3D Cephalometrics). Using placed fiducials, the module allows users to compute 2D angles: Yaw, Pitch and Roll; and decompose the 3D distance into the three different components: R-L , A-P and S-I. It is possible to compute the middle point between two fiducials and export the values.

Surface Registration

This module allows the user to compute and apply transformations (registration) between two 3D models (VTK file). The registration can be computed using the entire mesh of both models (Surface Registration), just a region of interest (ROI Registration) on each model, or two fiducial lists which can be projected on the meshes (Fiducial Registration).

Image Registration (NonGrowing)

This module allows the user to compute and apply transformations (registration) between two 3D volumes. The registration can use a one-label segmentation image to define a region of interest on each model.

Image Registration (Growing)

This module provides additional voxel-based registration options in addition to the functionality provided by the NonGrowing module.
